# 251_software_engineer_project

Repository dự án môn công nghệ phần mềm Frontend
## Công nghệ sử dụng
- **Docker-compose**: Setup
- **Frontend:** Giao diện Vite + Typescript
## Yêu cầu
- Cài volta: https://volta.sh
- Cài yarn
```bash
volta install yarn
```
## hướng dẫn setup project
- Git clone repository này về.
```bash
cd 251_software_engineer_project
yarn install
```
- Để chạy server
```bash
yarn run dev
```
