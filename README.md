# 251_software_engineer_project

Repository dự án môn công nghệ phần mềm Frontend
## Công nghệ sử dụng
- **Docker-compose**: Setup
- **Frontend:** Giao diện Vite + Typescript
## hướng dẫn cài đặt
- Git clone repository
- Chạy lệnh sau để setup
```bash
cd 251_software_engineer_project
yarn install
```
- Để chạy server
```bash
yarn run dev
```
